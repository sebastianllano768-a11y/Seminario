# SeminarIA - Instrucciones del Proyecto

## Skills Disponibles

Las siguientes skills estan en `.claude/skills/` y DEBEN ser consultadas segun el contexto:

### Superpowers (`.claude/skills/superpowers/`)
- **using-superpowers.md** - Framework general: invocar skills ANTES de responder
- **brainstorming.md** - Lluvia de ideas estructurada antes de implementar
- **writing-plans.md** - Crear planes de implementacion detallados
- **test-driven-development.md** - TDD: test primero, codigo despues
- **systematic-debugging.md** - Debugging sistematico en 4 fases
- **executing-plans.md** - Ejecutar planes paso a paso con verificacion
- **verification-before-completion.md** - Verificar antes de decir "listo"

### Code Simplifier (`.claude/skills/code-simplifier/`)
- **SKILL.md** - Simplificar codigo: claridad, consistencia, mantenibilidad

### Sequential Thinking (`.claude/skills/sequential-thinking/`)
- **SKILL.md** - Razonamiento secuencial para problemas complejos

## Reglas del Proyecto

- Idioma de comunicacion: Espanol
- Stack: Node.js backend, HTML/CSS/JS frontend
- Seguir las skills de superpowers segun el contexto de la tarea
- No implementar sin plan aprobado para cambios grandes
- Verificar todo antes de reportar como completado
- Codigo limpio y simple sobre codigo "inteligente"
